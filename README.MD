# react + egg 项目管理系统
- 前端：react + redux + react-router-dom + axios + antd
- 后端：egg + mysql

## 计划
```
├── 登录注册
├── 工作台
├── 项目管理
├── 系统管理
│   ├── 用户
│   └── 权限
└── 消息
```
